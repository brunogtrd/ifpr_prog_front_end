Convenções de Nomenclatura para Classes CSS

1. Use nomes descritivos

Escolha nomes descritivos que indiquem claramente o propósito da classe. Por exemplo, em vez de usar uma classe genérica como para um elemento que é uma caixa de destaque, use algo mais específico como highlight-box.

2. Use nomes simples e curtos

Evite usar nomes muito longos para suas classes, pois isso pode tornar seu código difícil de ler e manter. Em vez disso, escolha nomes simples e curtos que ainda sejam descritivos. Por exemplo, em vez de usar featured-article-with-image como nome de classe, use featured-article e adicione uma classe adicional como has-image para indicar que o artigo tem uma imagem em destaque.

3. Use letras minúsculas e hífens para separar palavras

Ao nomear suas classes, use letras minúsculas e hífens para separar as palavras. Evite usar espaços ou outros caracteres especiais, pois isso pode causar problemas de compatibilidade entre navegadores e tornar o código mais difícil de ler. Por exemplo, em vez de usar featuredArticle ou featured_article, use featured-article.

4. Seja consistente

Mantenha a consistência em suas convenções de nomenclatura para que seus estilos sejam fáceis de ler e entender. Use os mesmos padrões de nomenclatura em todo o seu projeto e evite misturar convenções diferentes.
