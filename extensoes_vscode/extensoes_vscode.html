Extensões do Visual Studio Code

Code Runner

Permite executar código em várias linguagens diretamente do editor de texto, sem precisar abrir um terminal.

Prettier

Formata o código automaticamente, seguindo as convenções do idioma escolhido. É possível configurar as regras de formatação.

Saiba mais

Live Server

Cria um servidor local para testar páginas HTML e CSS. Quando o arquivo é salvo, a página é atualizada automaticamente no navegador.

Saiba mais

GitLens

Integra o Git ao Visual Studio Code, exibindo informações sobre as mudanças no código e nos arquivos.

Saiba mais

Auto Rename Tag

Permite que você renomeie automaticamente a tag HTML de abertura ou fechamento quando você renomeia uma delas.

Saiba mais

Colorize

Destaca as cores nos arquivos de código-fonte para facilitar a identificação e leitura do código.

Saiba mais

Live Preview

Atualiza a visualização do navegador em tempo real conforme você digita o código HTML e CSS.

Material Icon Theme

Adiciona ícones de material design aos arquivos e pastas do VS Code para facilitar a identificação.
